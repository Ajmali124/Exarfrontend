## üí° DESIGN TIPS FOR POSTERS

### Light Theme Posters
- Use **#FFFFFF** as base background
- Add subtle gradients from **#F0FDF4** (30% opacity) to **#F0FDFA** (30% opacity)
- Primary accent: **#22C55E** (Green) or **#14B8A6** (Teal)
- Text: **#111827** for headings, **#4B5563** for body
- Borders: **#E5E7EB** for subtle separation

### Dark Theme Posters
- Use **#0A0A0A** as base background
- Add purple/indigo gradients: **#581C87** (20% opacity) to **#312E81** (20% opacity)
- Primary accent: **#A855F7** (Purple) or **#C084FC** (Light Purple)
- Text: **#FFFFFF** for headings, **#D1D5DB** for body
- Borders: **#262626** for subtle separation

### Color Combinations
- **Light Theme**: Green (#22C55E) + Teal (#14B8A6) on White (#FFFFFF)
- **Dark Theme**: Purple (#A855F7) + Indigo (#312E81) on Black (#0A0A0A)
- **High Contrast**: Use accent colors with their respective backgrounds for maximum visibility

## üìê OPACITY VALUES

When using RGB/RGBA:
- **80% opacity**: Use `CC` in hex (e.g., `#FFFFFFCC`)
- **50% opacity**: Use `80` in hex (e.g., `#FFFFFF80`)
- **30% opacity**: Use `4D` in hex (e.g., `#FFFFFF4D`)
- **20% opacity**: Use `33` in hex (e.g., `#FFFFFF33`)
- **10% opacity**: Use `1A` in hex (e.g., `#FFFFFF1A`)
- **5% opacity**: Use `0D` in hex (e.g., `#FFFFFF0D`)

Example:
- `bg-white/80` = `#FFFFFFCC` or `rgba(255, 255, 255, 0.8)`
- `bg-white/20` = `#FFFFFF33` or `rgba(255, 255, 255, 0.2)`
- `bg-white/5` = `#FFFFFF0D` or `rgba(255, 255, 255, 0.05)`
